# 📱 Dopamine Free Launcher

Um launcher Android minimalista projetado para reduzir distrações e ajudar você a focar no que realmente importa.

## 🧠 Ideia do Projeto

Vivemos em um mundo cheio de estímulos digitais. O **Dopamine Free Launcher** foi criado com o objetivo de eliminar distrações visuais e comportamentais, proporcionando uma experiência mais limpa, intencional e produtiva no uso do celular.

## 🚀 Funcionalidades

- Interface extremamente minimalista
- Redução de estímulos (cores, ícones chamativos, etc.)
- Foco em produtividade e uso consciente
- Experiência fluida com Jetpack Compose

## 🛠️ Tecnologias Utilizadas

- Kotlin
- Jetpack Compose
- Android Studio
